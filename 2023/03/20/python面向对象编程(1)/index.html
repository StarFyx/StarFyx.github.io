python 和 C++ 中类的区别

python中类也是面向对象的，把需要的函数或者模块看做一组对象的集合

特征 python C++
类的本质 类是一个抽象的对象，类本身也是对象 类是一段抽象的代码段，类不是对象
类的继承 支持多继承，可以从多个父类继承属性和方法 支持单继承和多重继承，可以从一个或多个父类继承属性和方法
类的访问控制 没有严格的私有、公有、保护等访问控制符号，只能通过约定俗成的下划线来表示私有属性或方法 有private、public、protected等访问控制符号，可以明确地限制属性或方法的可见性
类的构造函数和析构函数 构造函数是__init__方法，析构函数是__del__方法，都需要显式地定义和调用 构造函数和析构函数与类同名，都会自动地定义和调用

python中类的定义

1.定义的格式为 关键字class + 类名 + ：
2. init 前后分别有两个下划线
3. init的第一个参数永远是self，表示创建实例，在之后使用Person类的时候不用管self直接写入其它的参数即可

类对象的两种操作方式

属性化引用

属性引用 使用 Python 中所有属性引用所使用的标准语法：obj.name 有效的属性名称是类对象被创建时存在于类命名空间中的所有名称

  • MyClass.i 返回整数对象
  • MyClass.f 返回字符串对象
  • 对象也可以重新赋值进行修改

实例化

  • 创建实例化的person类带上参数 person = Person("Alice")
  • 利用创建的对象进行函数调用 person.greet()

用类管理时无需知道内部细节，也可以随时增加类内的方法
