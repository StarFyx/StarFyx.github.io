特殊字符

一些特殊字符需要用相应的代码来表示，如：<br> 单独打字会认为是markdown语言

特殊字符 对应代码
< &lt;
> &gt;
& &amp;
&times; &times;
&divide; &divide;
&sup2; &sup2;
&plusmn; &plusmn;

Markdown 使用了很多特殊符号来表示特定的意义，如果需要显示特定的符号则需要使用转义字符，Markdown 使用反斜杠转义特殊字符
Markdown 支持以下这些符号前面加上反斜杠来帮助插入普通的符号：
#
\
*
`
_
()
