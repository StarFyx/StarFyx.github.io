Github博客搭建

Posted on 2023-02-17

github主页注册

  1. 新建仓库，名称为starfyx.github.io，有大写的用户名要变成小写
  2. 在仓库里面新建index.xml文件
<!DOCTYPE html>
<html>
<body>
<h1>Hello World</h1>
<p>I'm hosted with GitHub Pages.</p>
</body>
</html>

访问域名即可

本地环境依赖安装

1. Git安装

官网下载 https://git-scm.com/
安装

2. node.js安装

官网安装https://nodejs.org/zh-cn/

git --version
node --version
npm --version

3. Hexo安装

npm install hexo-cli -g
hexo version

搭建并配置到github上

  1. 本地创建hexo源码文件夹
  2. 进入文件夹安装npm和插件
$ hexo init starfyx.github.io
INFO  Cloning hexo-starter https://github.com/hexojs/hexo-starter.git
INFO  Install dependencies
INFO  Start blogging with Hexo!

cd starfyx.github.io/
npm install

#安装部署插件
npm install hexo-deployer-git -save
  1. 启动本地验证
hexo server -o
  1. 更改根目录的config.yaml
deploy:
  type: git
  repo: git@github.com:StarFyx/starfyx.github.io.git
  branch: main
  1. 同步到github上
源码编译为静态网页文件
hexo generate
部署到github上
hexo deploy
  1. 更新文章
    在本地文件夹 source/_posts 中添加博客markdown文件，重复步骤5

更换hexo主题

  1. 更换主题next
    github主页https://github.com/theme-next/hexo-theme-next

  2. 进入themes克隆主题

  1. 更改yaml文件
    在_config.yaml 中找到主题，换为theme: hexo-theme-next

参考博客：https://oceanwang.top/personal-website-1/#!

FYX@DESKTOP-JC8V593 MINGW64 /d/Code/GithubPage/starfyx.github.io
