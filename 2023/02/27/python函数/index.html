2 函数参数

对输入参数检查

使用python函数： isinstance(变量名，检查的类型) 返回TRUE FALSE

默认参数

默认参数在不输入的时候使用默认值，也可以另外指定输入值

可变参数

用于输入不确定的变量个数，用加*的方式传入，相当于接受的是一个tuple

关键字参数

在接受时候相当于转化为了dict

==参数定义的顺序必须是：必选参数、默认参数、可变参数、命名关键字参数和关键字参数==

可变对象和不可变对象

在 python 中，strings, tuples, 和 numbers 是不可更改的对象，而 list,dict 等则是可以修改的对象

  • 不可变类型：类似 c++ 的值传递，如 整数、字符串、元组。如fun（a），传递的只是a的值，没有影响a对象本身。比如在 fun（a）内部修改 a 的值，只是修改另一个复制的对象，不会影响 a 本身。

  • 可变类型：类似 c++ 的引用传递，如 列表，字典。如 fun（list1），则是将 list1 真正的传过去，修改后fun外部的list1也会受影响

变量作用域

全局变量作用全局，局部变量作用域在函数内部，和其它语言相同
