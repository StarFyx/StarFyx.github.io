python 和 C++ 中类的区别

python中类也是面向对象的，把需要的函数或者模块看做一组对象的集合

特征 python C++
类的本质 类是一个抽象的对象，类本身也是对象 类是一段抽象的代码段，类不是对象
类的继承 支持多继承，可以从多个父类继承属性和方法 支持单继承和多重继承，可以从一个或多个父类继承属性和方法
类的访问控制 没有严格的私有、公有、保护等访问控制符号，只能通过约定俗成的下划线来表示私有属性或方法 有private、public、protected等访问控制符号，可以明确地限制属性或方法的可见性
类的构造函数和析构函数 构造函数是__init__方法，析构函数是__del__方法，都需要显式地定义和调用 构造函数和析构函数与类同名，都会自动地定义和调用

python中类的定义

1.定义的格式为 关键字class + 类名 + ：
2. init 前后分别有两个下划线
3. init的第一个参数永远是self，表示创建实例，在之后使用Person类的时候不用管self直接写入其它的参数即可

类对象的两种操作方式

属性化引用

属性引用 使用 Python 中所有属性引用所使用的标准语法：obj.name 有效的属性名称是类对象被创建时存在于类命名空间中的所有名称

  • MyClass.i 返回整数对象
  • MyClass.f 返回字符串对象
  • 对象也可以重新赋值进行修改

实例化

  • 创建实例化的person类带上参数 person = Person("Alice")
  • 利用创建的对象进行函数调用 person.greet()

python元组、字典、模块

Posted on 2023-03-02

元组

  1. 元组在初始化时一旦确定就无法更改，因此没有append等函数，其他和list类似
  2. 用小括号（）表示
  3. 空元素定义 my_tuple = ()
  4. 只定义元素个数 my_tuple = (2)
  5. 定义一个元素加上逗号 my_tuple = (2,)
  6. 内置函数
    比较不同元组 cmp(tuple1, tuple2)
    长度 len(tuple1)
    列表转化为元组 tuple(my_list)

字典

使用key-value的方式储存数据，达到快速查找的目的，类似于map

  1. key值不可变，不能用列表
  2. 同一个字典内的key不能重复

模块

  1. 将自己写的函数和功能进行封装，方便调用
  2. 调用第三方或者其它的库文件

自己的模块

  1. 放在统一文件夹的目录下
  2. 一定要有_init_.py文件
  3. 文件名称不要和系统内的包或者库重复
  4. 按照层级结构import 例如 import mylib.fyx

默认参数

默认参数在不输入的时候使用默认值，也可以另外指定输入值

可变参数

用于输入不确定的变量个数，用加*的方式传入，相当于接受的是一个tuple

关键字参数

在接受时候相当于转化为了dict

==参数定义的顺序必须是：必选参数、默认参数、可变参数、命名关键字参数和关键字参数==

可变对象和不可变对象

在 python 中，strings, tuples, 和 numbers 是不可更改的对象，而 list,dict 等则是可以修改的对象

  • 不可变类型：类似 c++ 的值传递，如 整数、字符串、元组。如fun（a），传递的只是a的值，没有影响a对象本身。比如在 fun（a）内部修改 a 的值，只是修改另一个复制的对象，不会影响 a 本身。

  • 可变类型：类似 c++ 的引用传递，如 列表，字典。如 fun（list1），则是将 list1 真正的传过去，修改后fun外部的list1也会受影响

变量作用域

全局变量作用全局，局部变量作用域在函数内部，和其它语言相同
